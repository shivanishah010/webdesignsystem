Text Styles

H1 Main Page Heading will be written in Arvo font size 44px, weight normal.

H2 sub-headings will be written in Arvo font size 36px, weight normal.

H3 sub-headings will be written in Arvo font size 28px, weight normal.

Body text will be written in Lato font size 18px, weight normal.

Photo captions will be written in Lato font size 16, weight normal.

Quotes will be written in Lato font size 18, weight normal. They will be differentiated from paragraph text with a solid claret-colored left side border of 4px.
